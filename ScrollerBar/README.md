## jQuery滚动条插件
jQuery滚动条插件，支持横向、纵向滚动；

#### 参数配置：
```javascript
foregroundColor: 'red',			// 滚动条前景颜色
backgroundColor: '#d5d5d5',		// 滚动条背景颜色
height: 400,					
width: 600,						
sliderBoxWidth: 10,			// 滚动条宽度 
sliderBarHeight: 100,			// 滚动条高度
mousewheel: true,			// 鼠标滚动
radius: 5,				// 滚动条圆角
scrollDirection: "y",			// 滚动方向，默认为纵向
```
