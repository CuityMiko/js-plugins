## 原生Js钟表插件
原生Js钟表插件，面向对象，分别用Canvas、Div实现；

#### 参数配置：
Div 版：
```javascript
bgColor: '#fff',
size: 300,
hColor: '#555',			// 时针颜色
mColor: 'gold',			// 分针颜色
sColor: 'orange',		// 秒针颜色
boxShadow: 'inset 0 0 60px #999',
cBgcolor: '#E91E63',
```
Canvas 版：
```
bgColor: '#fff',
size: 300,
hColor: '#555', // 时针颜色
mColor: '#f00', // 分针颜色
sColor: 'orange', // 秒针颜色
cBgcolor: '#e91e63', // 圆点颜色
markColor: 'green', // 外圈颜色
markSmallColor: '#333', // 小刻度颜色
lineWidth: 2     // 外圈线宽度
```
