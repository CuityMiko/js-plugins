## 原生Js钟表插件
原生Js钟表插件，面向对象；

#### 参数配置：
```javascript
bgColor: '#fff',
width: 300,
height: 300,
hColor: '#555',			// 时针颜色
mColor: 'gold',			// 分针颜色
sColor: 'orange',		// 秒针颜色
boxShadow: 'inset 0 0 60px #999',
cBgcolor: '#E91E63',
```
