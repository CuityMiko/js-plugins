## EasyDialog弹窗插件
Dialog弹窗插件，加入css3效果，支持拖拽，支持iframe，配置多样化；
#### 参数配置：
```javascript
width: 300,
height: 400,
borderRadius: 0,
boxShadow: '0 3px 15px rgba(0, 0, 0, .4), 0 0 5px rgba(0, 0, 0, .4)',
msg: 'Hello,this ia a eaeyDialog !',
title: '标题',
type: 'alert',
seconds: 5,
effect: 'default',
buttons: ['确定'],
buttonClass: ['danger', 'warning', 'success', 'primary', 'default'],
content: '我是被弹出的内容',
callback: function() {},
isDrag: false
```
